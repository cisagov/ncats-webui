Welcome to NCATS: Cyber Hygiene

Change Log:


  • 18-April-2018

    • Added new hire dashboard.
    • New checkbox in preferences to show only election stakeholders data on the dashboard.

  • 19-December-2016

    • Disconnects are now handled more discreetly. Gear turns red; no dialogue box.
    • Data compression is enabled to reduce bandwidth utilization.
    • Auto-rotate is enabled by default. This can be changed via a configuration file.
    • The interface exposed is now configurable per deployment (via a configuration file).
    • All data-feed connections are now displayed in the preferences screen.
    • App updates will force all clients to reload the application. This reduces human intervention for long running displays.

  • 6-May-2016

    • Improved accuracy of 'Stakeholder' and 'Reports Delivered' metrics on main Dashboard. Also replaced 'Open Ports' metric with more useful 'Vulnerable Hosts' metric.
    • Added new Closed Tickets CSV download button to Cyber Exposure dashboard.

  • 1-March-2016

    • Added Cyber Exposure dashboard to replace BOD dashboard.

  • 8-December-2015

    • Added preferences screen to allow configuration of site settings.
    • Added ability to obscure stakeholder names on the dashboard.
    • Added ability to cycle between the dashboard and maps.
    • Changed map layouts to better fit the screen.

  • 30-July-2015

    • Added a new ticket age graph to the BOD dashboard.
    • Added a draft fullscreen dashboard. This dashboard is being designed for use on a large monitor, and may not render optimally on mid-sized displays.
    • Converted data feeds to use socket.io. Additional conversions to follow.
    • Refactored pages so that they are recomposable. This allows BOD graphs to used both on the BOD page, and the new dashboard.

  • 29-June-2015

    • All maps now scale well with the browser window and smaller device screens.
    • A new full-screen button ( ​ )​ is available on supported devices.

  • 4-June-2015

    • The menu bar and item locations have been reorganized.
    • A new map of identified hosts in the U.S. has been added.
    • The Weekly Tickets Report has been added.
    • A severity filter has been added to the Active Vulnerabilities map.
    • Fixed an issue with the initial content of the Scan Progress page matching its filter.
